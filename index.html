Summary: Latitude vs X.

This project focused on analyzing weather changes as you go closer to the equator. We brought in data using OpenWeatherMap API, and surveyed over 500 cities. The technology used included Python, API, and JSON requests.

We used Matplotlib in order to plot each of our hypothesis. We looked at tempature, cloudiness, humidity, and windspeed as measurment points.
